# CRUD App

This app will do a simple create/read/update and delete operation from a simple webpage.

## Topics we will cover:

### Database

* Database setup using docker
* Schema/Table Creation
* Manipulate data from Pycharm

### Python App

* Connects to db
* Expose an endpoint (API) for CRUD

### HTTP Calls

* Use pycharm to make http calls to api endpoint

### Front-end

* Simple html table to do CRUD

**Introduction**

* Statements & Comments
* Python Variables
* Python Datatypes

**Python Flow Control**

* Python Functions
* Function Argument
* Python Modules
* Python Package

**Python Datatypes**

* Python Numbers

* Python Tuple
* Python String
* Python Set
